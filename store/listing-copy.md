## Hosting the privacy policy on GitHub Pages (free, 5 min)

Play Store **requires** a public URL for your privacy policy. The simplest free path:

1. Create a free GitHub account if you don't have one (https://github.com/signup)
2. Click "New repository" → name it something like `word-learner-privacy`
3. Make it **Public**, check "Add a README"
4. Click "Add file → Upload files" and drag in `store/privacy-policy.html`
5. **Rename it to `index.html`** before committing (so the URL is cleaner)
6. Commit
7. Go to **Settings → Pages** → set **Source** to "Deploy from a branch", pick `main` branch and `/ (root)` folder, save
8. Wait ~1 minute. Your URL will be `https://<your-github-username>.github.io/word-learner-privacy/`
9. Paste that URL into Play Console's "Privacy policy" field

If you want a custom domain later, GitHub Pages supports it for free.
