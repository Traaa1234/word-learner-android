# Word Learner

A colorful flashcard app that teaches the **200 most common English words** to early readers and ESL learners. Built with Capacitor — a single HTML/CSS/JS source builds to native **Android** *and* **iOS** apps.

## Features

- **200 top English words** ranked by frequency
- **Two modes**: flashcards and fill-in-the-blank
- **9 grammar categories** (nouns, verbs, adjectives, adverbs, pronouns, prepositions, determiners, conjunctions, auxiliaries) with color-coded badges and per-category filters
- **Phonetic respelling** + IPA notation for every word
- **On-device speech synthesis** via the `@capacitor-community/text-to-speech` plugin — works offline once installed
- **Voice picker** that auto-selects a female voice when available
- **Star counter, friendly fox mascot, animated card transitions**
- **No ads, no tracking, no analytics, no accounts**, COPPA / GDPR-K compliant

## Quick start

```bash
# Install dependencies
npm install

# Iterate on the HTML in a browser
python -m http.server 8080 --directory www
# → http://localhost:8080

# Sync changes into the Android project
npx cap sync android

# Build & run on a connected device or emulator
npx cap run android
```

## Project layout

```
www/                    Source — single index.html + inline CSS/JS
android/                Generated by `npx cap add android` (commit-friendly)
assets/                 Icon source PNGs (1024×1024)
scripts/                Python + Node helpers (icons, screenshots, feature graphic)
screenshots/            Play Store screenshots (generated)
store/                  Play Store listing copy, privacy policy, testing guide
```

## Build

```bash
# Debug APK (sideload-friendly)
cd android && ./gradlew assembleDebug
# → android/app/build/outputs/apk/debug/app-debug.apk

# Signed release AAB (Play Store)
cd android && ./gradlew bundleRelease
# → android/app/build/outputs/bundle/release/app-release.aab
```

The release build expects `android/keystore.properties` and `android/app/release.keystore` (both gitignored). Generate your own keystore with `keytool` if you fork this.
